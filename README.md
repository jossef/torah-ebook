# Torah E-Books

A personal collection of Hebrew prayer e-books and other Jewish religious texts in PDF
 format, organized for use on Kindle and similar e-book readers.

I created this collection while searching for well-organized Hebrew religious material for the Kindle I bought for my wife. I am sharing it here to make these resources easier for others to find and use.
